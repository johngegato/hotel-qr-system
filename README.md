\# Hotel QR Ordering \& Audio Communication System



A three-tiered hospitality platform enabling WebRTC-based audio calling, real-time queuing, and dynamic food/spa ordering.



\## Architecture



\- \*\*Guest Web App\*\*: QR-triggered web interface for guests

\- \*\*Front Desk Android App\*\*: React Native app for staff

\- \*\*Admin Dashboard\*\*: Web-based management panel

\- \*\*Audio Signaling Server\*\*: Node.js + Socket.io backend



\## Tech Stack



\- Frontend: Next.js, React Native

\- Backend: Node.js, Express, Socket.io

\- Database: PostgreSQL (Supabase)

\- Hosting: Vercel (frontend), Oracle Cloud (audio server)
